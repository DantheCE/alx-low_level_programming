# ALX Low-Level Programming

This repository contains programs written for the low-level programming and algorithm track of the ALX Software Engineering Program. The focus is on mastering C programming, memory management, and system-level concepts.

## Description

The projects in this repository cover a range of fundamental concepts in C:
- Data types, variables, and operators.
- Control flow (loops and conditionals).
- Functions and scoping.
- Pointers and memory addressing.
- Arrays and strings.
- Recursion.
- Static and dynamic libraries.
- Memory allocation (`malloc`, `free`).
- Data structures (linked lists, stacks, queues, trees, hash tables).

## Technologies

- **OS:** Ubuntu 20.04 LTS (WSL, Virtual Machine, or Native)
- **Compiler:** `gcc` (version 4.8.4 or higher)
- **Style:** All C files are written in compliance with the Betty style (standard for ALX/Holberton).

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`.
- All files are compiled on Ubuntu 20.04 LTS using `gcc`, using the options `-Wall -Werror -Wextra -pedantic -std=gnu89`.
- All files should end with a new line.

## Usage

To compile and run any program, navigate to the specific directory and follow the instructions in its `README.md`. Generally:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 main_file.c source_file.c -o output_name
./output_name
```
